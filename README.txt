Simple BrainFuck Compiler

About
This project allows to run BrainFuck codes which consist of
seven commands - + - > < [ ] .
More about BrainFuck - https://ru.wikipedia.org/wiki/Brainfuck

Features
- Compiling and run of the program are separated into to different steps
Compile class is responsible for compiling and Invoker is responsible for running.
- Different unit and functional tests added to the project.
